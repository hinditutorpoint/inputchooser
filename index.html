üöÄ Basic Usage

HTML Setup

<div class="dropzone-area" 
    data-name="file_upload" 
    data-size="1024" 
    data-type="image/*">
</div>

JavaScript Initialization

document.addEventListener("DOMContentLoaded", function() {
    new InputChooser(".dropzone-area");
});
Live Example

‚öôÔ∏è Configuration Options

HTML Data Attributes

Attribute Description Default Example
data-name Input name attribute for form submission Required data-name="profile_image"
data-size Maximum file size in KB 10240 (10MB) data-size="2048" (2MB)
data-type Allowed file types (MIME types or extensions) Common file types data-type="image/*,.pdf,.docx"
data-width Maximum image width in pixels None data-width="800"
data-height Maximum image height in pixels None data-height="600"

JavaScript Options

new InputChooser(".dropzone-area", {
    maxFiles: 5,          // Maximum number of files allowed
    multiSelect: true,     // Allow multiple file selection
    customExtensions: ['.psd', '.ai']  // Additional file extensions to accept
});
